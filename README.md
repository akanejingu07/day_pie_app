# MY DAY – 1日の過ごし方可視化アプリ

1日の行動を「分」単位で入力し、  
1440分（24時間）の使い方を円グラフで可視化できるWebアプリです。

スマートフォンでも使いやすいUIと、  
ドラッグ操作による直感的な並び替えを重視して開発しました。

---

## アプリ概要

- 行動名と所要時間（分）を入力
- 入力した行動が一覧に表示
- 合計1440分に近づくほど円グラフが埋まる
- 残り時間は薄いグレーで表示
- 行動はドラッグ操作で並び替え可能
- スマホ・PC両対応のレスポンシブデザイン

## 機能一覧

- 行動の追加
- 行動の削除
- 行動の並び替え（ドラッグ操作）
- 円グラフによる時間配分の可視化
- 残り時間（1440分との差分）表示
- スマホ対応レスポンシブUI

---

## 工夫したポイント

- 1440分という制約を視覚的に理解できるUI
- 円グラフのはみ出し問題を考慮した  
  スマホSafari対応のレイアウト設計
- ドラッグ用ハンドル（≡）を設け、  
  誤操作を防ぎつつ直感的な並び替えを実現
- 入力・一覧・グラフをシンプルに分離し、  
  毎日使っても疲れないデザインを意識

---

## レスポンシブ対応

- スマートフォン：縦スクロール前提の1カラム
- PC：余白を活かした見やすいレイアウト
- タッチ操作でもドラッグしやすいUI設計

---

## 注意事項

- 本番環境では SQLite を使用しています  
  （Render無料プランの仕様上、再デプロイ時にデータはリセットされます）
- 個人利用・ポートフォリオ用途を想定しています
